## 不使用递归，怎么高效写出阶乘表达式

通过 reduce 和 operator.mul 函数计算阶乘
```
>>> from functools import reduce
>>> from operator import mul
>>> def fact(n):
    return reduce(mul, range(1, n+1))

>>> fact(5)
120
```
