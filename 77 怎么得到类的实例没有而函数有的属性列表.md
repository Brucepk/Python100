## 怎么得到类的实例没有而函数有的属性列表

创建一个空的用户定义的类和空的函数，计算差集，然后排序。

```
>>> class C:
    pass

>>> obj = C()
>>> def func():
    pass

>>> sorted(set(dir(func)) - set(dir(obj)))
['__annotations__', '__call__', '__closure__', '__code__', '__defaults__', '__get__', '__globals__', '__kwdefaults__', '__name__', '__qualname__']
```
