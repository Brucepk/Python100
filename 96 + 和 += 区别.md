## + 和 += 区别

+ 两边必须是同类型的对象才能相加，+= 右操作数往往可以是任何可迭代对象。

```
>>> l = list(range(6))
>>> l += 'qwer'
>>> l
[0, 1, 2, 3, 4, 5, 'q', 'w', 'e', 'r']
>>> j = l + (6, 7)
Traceback (most recent call last):
  File "<pyshell#91>", line 1, in <module>
    j = l + (6, 7)
TypeError: can only concatenate list (not "tuple") to list
```
